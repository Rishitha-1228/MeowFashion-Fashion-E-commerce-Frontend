## 📌 Project Description
This is a full-stack fashion e-commerce web application where users can browse products. The frontend is built using HTML, CSS, and JavaScript, and the backend is built using Node.js and Express.

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Deployment: Render
- Version Control: Git & GitHub

---

## ✨ Features
- 🛒 Display products from backend API
- 📱 Responsive design (mobile-friendly)
- 🔄 Dynamic data fetching using JavaScript
- ⚡ Fast loading UI
- 🧾 REST API for product management
